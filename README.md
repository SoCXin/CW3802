CPU：高工作频率在80MHZ，工作频率可调在100KHZ到80MHZ,在flash运行能做到真正无等待工作频率32Mhz,CPU 在内部RAM运行能到真正无等待工作频率80Mhz。

* 64KByte Flash
* 16KByte SRAM
* 256Byte EEPROM
